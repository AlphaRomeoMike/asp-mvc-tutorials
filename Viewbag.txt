Passing data from Controller to View
	We can use 
		Viewbag,
		Viewdata,
		TempData

ViewBag passes data from controller to view:
	Dynamic keyword can allocate any type of data,
	char, float, int, float, double etc
	No type casting is required for this
	It is a dynamic property
	Does not require typecasting
	Syntax:
	ViewBag.<PropertyName> = <Value>
	Datacan be shared between ViewData and View Bag
